README

simpletoc

Simple Table of Contents generator for DOM and Markdown. 1.5kb

Quick dive

For the DOM

simpletoc.domToc({
  // Only search for content in a specific place.
  root: '.content',
  // Only use headings from 1 to 3 levels.
  selector: 'h1, h2, h3',
  // Where to place the generated Table of Contents.
  target: '.table-of-contents-placeholder',
  // Use an unordered list (bullet list, no numbers).
  type: 'ul'
})

For Markdown

simpletoc.mdToc(markdownText, {
  // Place to inject the generated table of contents.
  target: /Table of contents here/,
  // Use an unordered list (bullet list, no numbers).
  type: 'ul'
})

Why?

I needed to generate a table of contents and all I found was kind of overkill or bulked.

Install

$ npm install simpletoc

Or grab a copy from the dist/ folder – there's no CDN available at the moment.

API

domList

Given a tree of DOM elements, creates a new list of anchors which point to the DOM elements.

domList(tree: Array, options: Object?): HTMLElement
Parameters
tree (Array) Array of nested arrays representing a tree.
options (Object? = {}) Options object.
Name Description
options.type string (default 'ol') Type of DOM element to use.
options.className (string | Array) (default 'simpletoc') Class(es) for the list.
options.getId Function? Function to generate an id.
options.getAnchor Function? Function to generate an anchor DOM element.
Returns
HTMLElement: A new HTMLUListElement.
Example
function createHeading (level, text, id) {
  return document
    .createRange()
    .createContextualFragment(`<h${level} id="${id}">${text}</h${level}>`)
    .firstElementChild
}

const tree = [
  [createHeading(1, 'First', '1'), [
    [createHeading(2, 'Second', '2')]
  ]],
  [createHeading(1, 'Third', '3')]
]

domList(tree)
// => <ol class="simpletoc">
//      <li><a href="#1">First</a>
//        <ol class="simpletoc">
//          <li><a href="#2">Second</a></li>
//        </ol>
//      </li>
//      <li><a href="#3">Third</a></li>
//    </ol>

domToc

Handy function to automagically generate a table of contents based on the document and (optional) CSS selectors, and attach it somewhere in the DOM (by default to an element with the "simpletoc" attribute).

domToc(options: Object?): HTMLElement
Parameters
options (Object? = {}) Options object.
Name Description
options.compare Function? Function used to compare hierarchy of heading elements.
options.root string (default 'body') CSS selector to scope the search of headings.
options.selector string (default 'h1,h2,h3,h4,h5,h6') CSS selector to search for headings.
options.target string (default '[simpletoc]') CSS selector to seach for the element where the TOC is going to be attached.
Returns
HTMLElement: A new HTMLUListElement generated by #domlist .
Example
domToc()
// => <ol class="simpletoc">
//      <li><a href="#1">First</a>
//        <ol class="simpletoc">
//          <li><a href="#2">Second</a></li>
//        </ol>
//      </li>
//      <li><a href="#3">Third</a></li>
//    </ol>

mdList

Given a tree of markdown headings, creates a markdown list of references that point to those headings.

mdList(tree: Array, options: Object?, depth: number): string
Parameters
tree (Array) Array of nested arrays representing a tree.
options (Object? = {}) Options object.
Name Description
options.type string (default 'ol') Type of list to use, ordered or unordered.
options.getRef Function? Function to generate a markdown reference.
options.getText Function? Function to generate the display text of the reference.
depth (number = 0) Initial level of indentation.
Returns
string: A new markdown list.
Example
const tree = [
  ['# First', [
    ['## Second', [
      ['### Third']
    ]]
  ]],
  ['# Fourth', [
    ['## Fifth']
  ]]
]

mdList(tree)
// => "1. [First](#first)
//         1. [Second](#second)
//             1. [Third](#third)
//     2. [Fourth](#fourth)
//         1. [Fifth](#fifth)"

mdList(tree, { type: 'ul' })
// => "* [First](#first)
//         * [Second](#second)
//             * [Third](#third)
//     * [Fourth](#fourth)
//         * [Fifth](#fifth)"

mdToc

Handy function to automagically generate a table of contents for a markdown document.

mdToc(text: any, options: Object?): string
Parameters
text (any)
options (Object? = {}) Options object.
Name Description
options.compare Function? Function used to compare hierarchy of headings.
options.extract Funtion? Function used to extract headings from the text .
options.target RegExp? Regex to seach for text to be replaced with the table of contents.
Returns
string: The input text with a new markdown list generated by #mdlist attached on target .
Example
const mdText = `

TOC

# First

Bla bla Bla.

## Second

Bla.

# Third

End.
`

mdToc(mdText)
// => "
//
// 1. [First](#first)
//     1. [Second](#second)
// 2. [Third](#third)
//
// # First
//
// Bla bla Bla.
//
// ## Second
//
// Bla.
//
// # Third
//
// End.
// "

tree

Creates a tree of nested arrays based on arr data and fn criteria. Elements are compared against their left neighbour using fn which is invoked with these arguments: (current, next). If the invocation returns truthy, next will be nested into current, next will be placed at the same level of current otherwise.

tree(arr: Array, fn: Function): Array
Parameters
arr (Array) The array to structure as tree.
Name Description
arr.0 any
arr.rest ...any
fn (Function) The function invoked per item in the array.
Returns
Array: Returns an array of arrays representing the tree.
Example
const levels = [
  5,
  3,
  2,
  5,
  4
]

function comparator (current, next) {
  return next < current
}

tree(levels, comparator)
// => [ [ 5, [ [ 3, [ [ 2 ] ] ] ] ], [ 5, [ [ 4 ] ] ] ]
